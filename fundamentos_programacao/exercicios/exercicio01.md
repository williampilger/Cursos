# Exercício 01

## Enunciado

Faça um algoritmo que leia um valor e diga se ele é **par** ou **ímpar**.

## Solução

```py

numero = int(input("Digite o valor: "))
if(numero % 2 == 0):
    print("O número é par!")
else:
    print("O número é ímpar!")

```
